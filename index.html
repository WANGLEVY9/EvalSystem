EvalSystem

多轮对话指令遵循自动评测系统 — 可解释 · 可量化 · 可复算
通过用户模拟器、混合评估器、证据三元组、自一致性采样等机制，把"对话好不好"这个主观问题转化为可工程化的指标。

🚀 三步上手

1. 克隆仓库并安装依赖：
git clone https://github.com/WANGLEVY9/EvalSystem.git && cd EvalSystem && pip install -r requirements.txt

2. 验证安装（无需 API Key）：
python main.py --demo

3. 配置 API Key 后跑真实评测：
export DEEPSEEK_API_KEY="sk-..." && python main.py -i config/sample_instructions/rider_feimaotui.md --personas cooperative --seed 42
